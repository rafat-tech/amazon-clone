# Amazon Clone

This is a simple Amazon homepage clone built using **HTML and CSS**.

##  Features

- Header with logo and search bar
- Navigation bar
- Product display sections
- Footer section

## Technologies Used

- HTML
- CSS

##  Folder Structure

- `index.html` – Main HTML file
- `style.css` – Styling
- `images/` – Logos and product images
